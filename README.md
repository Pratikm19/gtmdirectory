# GTM Tech Stack Directory

A premium, interactive directory of curated GTM and RevOps tools.

## Tech Stack
- Vanilla HTML5
- Vanilla CSS3 (Custom Design System)
- Vanilla JavaScript
- JSON Data Source (`gtm-data.json`)

## Features
- Real-time filtering by Layer
- Full-text search across names and roles
- "Not Ideal If" strategic advice for every tool
- Priority tiering (Core vs Supporting)
- Premium dark-themed glassmorphism UI

## Deployment
This project is ready for static hosting on Cloudflare Pages, Netlify, or Vercel.
- **Entry point**: `index.html`
- **Assets**: `style.css`, `app.js`, `gtm-data-final.json`
